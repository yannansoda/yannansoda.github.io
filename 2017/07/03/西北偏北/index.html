西北偏北

2017-07-03
[Phlegethon灼焰]

西北偏北，或者偏西，2017年夏天，我在那里度过了难忘的半个月。在一路向东的长途火车上，我潦草地涂写下了这些笔记。   # 乌鲁木齐 我意识到这是一种并不恰当的方式，当从一个边疆的都市中心，迁徙到另一个边疆的都市中心：在史诗中，应当是去国离乡，携金银财宝出征，其后一路烟尘，财物挥霍一空，终于抵达另一个盛世，吃了苦，看见苍生，留下眼泪。 然而这太古典了。现代化的工具，比如说飞机，让你恨不得越快越好。越快越好，终于走下了晚点的飞机，我这么想着，毫无防备地被强烈的日光穿透。 我被太阳割得稀碎。时间稀碎，人群稀碎。在乌鲁木齐日不落的晚上，六点半如同正午，夜晚十点如同早晨，时间被碾碎了，随意抛一把，四下散落。傍晚八点，公交车仍弥漫着金色的烟尘，树荫透绿，被光调和过的颜色抹在树杈子上。人也不知走去了哪里，理应繁华的商业街上，行人散落，街道宽阔，因为车辆的稀落，立交桥像一条平坦的宽宽的带子。夜再深一些，天色暗了一些，日光慢慢汇聚着下沉的时候，人多了一点，却也只是一点点。十一点了，幼童尚未进入梦乡，夜晚似乎刚刚开始。 在乌鲁木齐的另一个问题是，我完全分不清哪里更像是乌鲁木齐。是游人攒动的大巴扎？千篇一律的旅游特产？抑或人来人往的中山路？目光能够触及的民族性，除了夹杂在人头中的花头巾，一双双一闪而过的明亮大眼睛，和盘曲在汉字之下的一行行维语之外，在草草扫视之中所剩无几。除了那些醇厚、筋道、超大块头的食物之外，除了丝裙、花巾、极浓的眉毛和眼线之外，除了天生的欧陆人的身材之外——除了这些丰富多彩的表面事物之外，其余的部分其实极单调。在这一点上乌鲁木齐与呼和浩特类似，有过之而无不及，近乎被改造没了个性：大型路口必然配着巨型广告牌，宣传着民族团结大一统云云，所有的公共场合都播放着同样的音乐，以宣传社会价值观。无论是当地人的巴扎，还是高档的百货商场，更不必说火车站、汽车站……处处是安检。安检人员或许是乌市最容易就业的岗位了。 我们只有两次瞥进了民族深处——一处是土耳其超市，它仿佛是坐落在攘攘游客中的另一个世界。另一次我们无意间闯入了维族人的生活区。躲在六车道背后的土路两侧是破破烂烂的小铺子。卖货的女人席地而坐，身下是斑斓的毛毯；一位卖瓜老汉守着一堆青黄的蜜瓜，孤零零地看着另一摊卖羊肉串的——那肉串真大。肉串摊前，一个维族小男孩迷惘地盯着摊前的食客，近乎面无表情。那条的土路的尽头便是一座清真寺，再转过去再能走上市区寻常的马路。这里终于没有了永不休止的《中国人宣言》。维语的流行音乐伴着烤羊肉串的烟，热烘烘地播放着。 还有天池。天池离乌鲁木齐不远，比想象中大得多，蓝得多，如梦如幻得多。尽管旅行团也多得多。穿过池子中心的道观，登上观景台，一道水波的褶充当景观的分界线，分界线的一侧是重重云嶂之下的湖绿色，另一侧是乍晴天空映出的碧蓝。我喜欢北疆山坡上的野花野草，明明在水草丰茂的山间，明明被层层密林包覆，却还是自顾自地开着，自顾自地窜高、窜得粗壮，自顾自地旋转着花瓣。在天池也是一样，穿过野蔷薇俯瞰山与水，像看一幅边框镶嵌了花朵的画。   # 北屯和布尔津 北屯是很不像城市的一座城市，我们从那里搭车去布尔津。北屯的新华书店不知何故已经关张，青年人却不见小城青年的窘态，人们漫不经心地乘车、买东西，在长满白花的花坛边发呆。这座城市的一半由兵团掌管；在荒郊野外，有一条“上海商业街”。 而布尔津是最不像小镇的小镇。布尔津使人平心静气。一面是专门为旅游而生的地理位置，众多的酒店，华丽得突兀的建筑；另一面则是稀少的人迹，宁静的景致。俄式的小房子仿若道具一般的摆设，几条宽敞的马路因为无人行驶而显得更加宽敞。炽烈的阳光给街头绿地和木椅铺满一层金色，几个做工粗劣的俄罗斯套娃天真地立在额尔齐斯河徐徐的波涛之外；河的对岸，森森的白桦树林是被装饰了的国界线。 初夏的布尔津已经极热，正值蚊虫猖獗的季节，人人头上都得罩几层纱网，否则滩地上的飞虫就会猛地钻进人的眼睛和鼻孔里。与密密麻麻的飞虫相比，布尔津的人太少了，人影零星地点缀在街道上。当然，安检机器旁的人口密度最高——后来发现，这在新疆的每个城市都是一样。中午时分，镇上的店铺多半都关张了，人们慢吞吞地挪动着，这一家的小姑娘轻悄悄地溜进另一家的店面里，包着头巾的女人又悄然消失在店面背后。广场和公园都被栅栏围着，其中空无一人。相比之下，布尔津的美食街显得极热闹，新疆餐馆和哈萨克餐馆相邻着，冒着油烟的摊子在街上，餐桌在街边的凹地里。骨碌着圆眼珠子的维族小孩子靠在墙边，咬着手里一只咬不完的烤包子。 在布尔津附近的五彩滩，傍晚，我包着头巾撞在温热的虫阵中。吊桥下的额尔齐斯河泛起了夕阳的浮光，隔着纱只能望见一片朦胧的风景，如旧时代的模糊照片。再往夕阳深处走，所有的红色，深的浅的偏橘的偏粉的，都混成一幅在奇诡画面上的景观。视野开阔，平坦的山脊之间只有人影和虫阵，霞光一层层地泛开。脚下踩着丹霞的花纹，在朦胧之中仿佛登上了另一个使人天旋地转的星球表面。   # 喀纳斯 还没到喀纳斯，我就已经觉得不真实了。去喀纳斯的路上是一处又一处草甸，一坡又一坡牛羊，草原如同地毯无穷无尽地铺展，森林则像整群整群地被复制过，粘贴在视野的边际上。真到了喀纳斯的群山之间，眼睛里只有雾，眼球被雾气粘住，从树林移到溪水间，又从溪流移回到草坡上。山的曲线远远望去都是平缓的，走近了细看，几道细细的雪线拖了下来：清白的雪，黛色的山。 走在通往观鱼台的石阶上，我意识到了古人只能靠咏叹来留存记忆的不易。我一个字也描述不出来，那样移步换景、瞬息万变的景观。一边是碧玉似的湖面，一边是野花开满山坡。一阵子是云雾缭绕，一阵子是蓝透了的天幕。往上头走，好似永远也走不到头——或者说，希望永远也走不到头。 喀纳斯湖变化万千，随心所欲。清晨全是迷雾中的森林，西伯利亚冷山在远远的河那头沉思，思虑也全都是雾。在迷雾里走着，杉树的针状叶挂着露珠，露珠凝重，轻微的触碰也不能使它降落。穿过迷雾，只消一阵子，树木从冥想中醒来，鸟也嚷嚷起来，水流有了呼吸，连植物也嚣张地竖起了刺，戾气被绿意所掩饰。再往前走，就走到了水边，阴天下泛绿的湖水在晴天渐渐变蓝，此刻也活泼起来，涌动着打旋儿的水涡。偶尔有鸭子轻松地浮在水面上。 而我觉得在喀纳斯湖边最好看的，是一匹马儿在流水边饮水的图景。我一张照片也没有拍到；我一直一直地记着，却再未与之重逢。一切好似都无法再重逢，因为太向往重逢。然而天气时景又如何重逢。喀纳斯是一个自行运转的系统，无意于观者的凝视——本来也是这样，它哪里会在意呢。 我们先在图瓦新村住了两天。老板的儿子既做招待，又做厨师，时常去换乘中心接人，还要冲到门口做防暴演习。身上的行头也在短袖、羽绒服、防暴背心和皮围裙之间迅速切换。村道里有牛。后来才知道，当地养牛受限，有这几头，已经是幸存。图瓦新村的夜雾是柔柔的蓝色，像是冷静的情人。 白哈巴很假。这是假的西北第一小镇，假的哈萨克斯坦界碑，假的木屋、山坡、图瓦人，假的牛奶桶子与皱纹。世界上怎么又这么虚假的地方。我是说，绝非美得不真实，也不比楚门的世界友善或簇新。这里只是虚假：人烟稀落，午后的阳光永不消褪，村子里只有四条路、四个方向，无论怎么走，都是已经被决定了的景观。 白哈巴的人像牛，反刍一样慢慢吞咽着奶疙瘩和油炸点心。他们像牛，长长的睫毛低垂，眼睛缓慢地眨着，执拗又迟钝的眼神坚定地流露出来，他们耸着高高的颧骨，踏着厚实的皮靴，像牛一样默默地缓步前行。跟他们说话也如对牛弹琴——全村，只有回族邮递员的句子我能迅速地听明白。傍晚，人们归家，牛被留在山坡上吃草。人们的院子点起炊烟，牛在夕阳余晖里甩着尾巴，和着慢拍。错过了白哈巴的晨雾，早晨一睁眼就看到院子里奔跑着一只小牛犊。在这里，人像牛一样；而此时这只奔跑的小牛，竟像只小狗。 前往禾木的车子驶过了大片草地与森林，仿佛经过了一本俄罗斯小说。而到了禾木，又说不上来它像什么地方。层层叠叠的木屋堆积在禾木河以东；河西则是相连的陡峭山峰，有骏马飞腾而上。沿着木栈道上山，山上是传说中成吉思汗点将的观景台，望下去只一片屋宇房梁，隐在桦树和杉树之间，左接山，右接水，山顶一片光雾弥漫。禾木河则是另一番动态——树林子有多静，河流就有多闹腾。湍流打着卷儿漂，河水在晚些时候的冷光下闪着蓝色，看起来便是薄荷味道。往回走的路上，村子陷入黑雾。我很久没有走在这样彻底的黑夜里，愈往黑处走，愈像进入封闭的空间。村子里没有路灯，所有的牛大概都已经睡了吧。 去看禾木的晨雾与炊烟的早晨迟了到，时间尴尬，日出已经终结它的辉煌，炊烟正陆续登场，还未被斜射的日光点亮。但我想了想，禾木就是禾木，哪里也不像，走在村里唯一的一条汽车道上，越接近禾木河，商铺越显得商业化，有些店铺已经放起了烂大街的流行音乐。炊烟是生活的日常，还是象征性的表演？我也不得而知。在堆满了长枪短炮的观景台上，野花最可爱，后面吃草的马匹最可爱，清秀的白桦林最可爱。人们太羡慕并不可爱的乡村生活。他们也并不可爱。但我依然喜欢禾木。   # 新疆 离开新疆之前又回到了乌鲁木齐，四次进出火车站，我几乎要记住了快速路的走向。乌鲁木齐站，大致是脱离乌鲁木齐市而存在的另一个区域。再回到这座日光城市，前夜大雨的湿气快要被蒸发干净，又是街道两旁的浓荫，又是郊外光秃秃的土疙瘩，又是一道又一道的安检。 喀纳斯景区里不允许大量畜牧，于是大片葱郁的草甸才得以保存。在禾木，许多新建的木屋却已经因为过度开发而面临着被拆除的命运，清晨七点，就有人骑在屋脊上拆房子。而乌市，没有我想象中的异族风情，印象深刻却是无处不在的安检、防暴措施中极度的谨慎、温吞的热情与不急不躁的个性。这座城市现代的一面与其他城市毫无区别，除了会耐心等待追车的人的公交车，除了懒洋洋的上班族，除了不紧不慢的吐字。 这些都是我的认知之外的北疆。新疆于我，早已存在于舌头乐队的暴烈中，存在于张智与旅行者的淡然中，存在于图瓦音乐的苍凉中，存在于刘亮程和李娟的文字里。我太向往它们给我的新疆烙印了，于是，非常轻易地，看见了这些烙印的反面。 此时在远离新疆的火车上，无法想象下一次归来在何时。然而正面也好，反面也罢，新疆像是大如脸盆的烤馕，像是石灰石一般的奶疙瘩，巨大，壮观，经得起咀嚼，便于携带。   # 西宁 西宁之大，超出预计。这里的“大”主要说的是大气。我就住在市中心，踩着清晨刚刚洒过水的路面，穿过复杂的地下通道，走入一片片百货大楼的丛林。 印象中的青海充满海子诗歌里那浪漫的贫穷与苍凉，西宁却有着满满的商铺，或新潮洋气，或狭小破旧，密密地堆砌在并不宽阔的道路两旁。除了商铺，便是饭馆。西北菜，清真饭馆，川菜，还有意想不到的海鲜。甜醅是青稞版本的酒酿，牦牛肉比普通牛肉更常见，红红的一碗辣子也很常见。漂着黄油的酸奶则出乎意料地甜。 商业街的一侧便是东关清真大寺。大寺的确优美。新建的门楼背后，是旧日的五拱门，处处是白色金色绿色，处处皆是曲线。汉人如我，只当它是一座历史悠久的建筑；对伊斯兰教徒而言，只是西北五省最有名望的大寺，常有人前来膜拜，至开斋节日，西宁及远近教徒溢出大寺庭院，街道都被占满——这些都是主殿门前一名教徒告诉我们的——一天要礼拜五次，他正在进行第三次祷告。大寺的殿门不对游人敞开，透过门缝，隐约能瞧见里面有国民党高官的题词，还挂着清风霁月的对联。大寺前的店铺里，挂满了信徒去朝圣的照片，那位热情的教徒说，西宁有许多人已经去过了麦加。 时值开斋，满街都是包着五颜六色头巾的回族女子。在街角的二楼向下望，全是一副副娇好的面容，美丽的头巾衬托着象牙白的皮肤、高挺的鼻梁和精致的杏仁眼。而走出大寺，又是满目白帽与花色头巾，消失在大寺墙外的一家家商铺中。   # 塔尔寺与青海湖 我总记得，杨晨说塔尔寺里有荷花池。真到了塔尔寺，我翻来覆去地看地图，没有，就是没有。塔尔寺有许多游客，还有比想象中多得多的寺院。从前，我只当它是一座小小的，诵经声无休无止的小寺庙。真正站在入口前，看见依山势而建的重重寺院，竟有看见了微缩版布达拉宫的错觉。 从前也进过许多藏传佛教的寺庙，但是在这儿，我第一次遇见磕长头的人。他们多是藏民，趴在地上做俯卧撑似地起起落落。我也第一次看见了木制纹刻门楣，在漆红的房门头上倏然暗下去一块。我还第一次听到了和着打击乐的经乐声，声响大作，僧人们团坐在大殿中央不言不语，任由绕行一周的游客无礼地打量。我第一次看到了酥油花制成的浮雕，却非为了宣扬教义或歌颂活佛，而只是为了描绘宗喀巴一个绮丽的梦。 再往下走，就是青海湖了。青海湖起初是天际远远的一条蓝线，有些孔雀绿掺杂的蓝。再往近行驶，终于一整个地从草地中浮出。那诗一点不假，青海湖，这宝石的尸体。除此之外，我不知如何形容这一汪蓝。穿过村民的油菜花地，来到湖边，牦牛和马匹乖顺地被牵着，我跟它们一起在湖边发呆。这一路走来，最想做牛羊，留在山与湖之间。 青海湖被草甸环绕着，等出了青海湖再往柴达木盆地走，植被逐渐稀薄。到了茶卡盐湖，只有雪地一般的盐湖和无云的晴空，除了游客之外，一片灰白。下水行走，盐粒硌得脚生疼，无数游人的衣袂翻飞，一片混乱。要我说，最好看的还是远处无法下水的盐湖中的几座盐堆，它们如同曝晒在灼热日光下的清冷的冰山。   # 祁连 往祁连山区，一路竟然都是草原与缓缓转过的群山。车子行驶在盘山公路上，常常独占一整条公路，只有山坡河谷间的羚羊与牦牛如黑白珍珠散落着。 我开始经过那些诗里的地方，德令哈——我们与去往德令哈的方向背道而驰；哈尔盖——哈尔盖是一段短短的路，我没有找到那里的火车站；最终是祁连——明月出天山，苍茫云海间…… 西宁更像是回族人的聚居地，到了青藏高原，才能意识到青海的藏民之众。开商店的往往是回族人，那些披挂着衣袍怯怯地买东西的，皆是藏民。无论是在黑马河镇，还是在哈尔盖乡，城里都只有几条直直的马路，路两旁大多是为游客而建的宾馆和饭馆，而馍馍店和酥油茶馆全挤在犄角旮旯里，一张张高原红的脸孔挪进去，黑洞洞的店铺里闪烁着一双双明亮的眼睛。 祁连县是一处干净漂亮的小城，道路都是大陡坡，县城对面就是卓尔山，夜晚的山间亮起了一盏盏昏黄的灯，仿佛有人守卫于此。祁连县的羊肉只要七十块钱一斤，祁连县的炒青菜总是满满一大盘，祁连县的商店晚上十点多了还不关门。 我从来不知道祁连有草原，更不知道这里还有丹霞。站在卓尔山顶，左侧深绿浅绿，右侧深朱浅褐，远处还有棱角分明的雪山。再往山上走去，云朵阴影下的远山如丝绒般泛着淡淡的光；往回去，日头高升，格子似的油菜花地和麦田便成了被点亮的马赛克拼贴。 祁连山下，我们路过了阿柔大寺。与塔尔寺相反，这寺虽以“大”而命名，却只有独独几幢，连着周围几片僧人的住处，孤独地占据着一小片山地。这里只有几个藏民信徒和数量同样稀少的游客。正值午饭时间，有人吹着螺号通知开饭，远处一个又一个着藏袍的老妪，以龟速挪腾过来。还有几个信徒，无视这号声，一圈圈地绕着寺院中央的毡房样建筑环行，每行一周，便在台阶上放一颗石子。圆顶房子旁边，有另一处开了一条小缝的小房间，大约有活佛高僧供奉其中，因此不时有人拜在门外。庭院正中，青青草丛围着大殿，门扇之上有牛皮帐子悬挂着，十分特别。转经筒下，彩色窗楣下，不时有披藏式衣袍的身影缓缓经过。我竟觉得，这里比塔尔寺更纯洁庄严。 行车路上，也常常有祭祀风马旗，五彩经幡在风中翻飞，二维码似的黑白经幡小纸片也在空中飘洒。这是一场信仰的风雨，降临在山间的垭口，往近了去，风雨兴起伴着咒语，从祭祀塔的扬声器中穿来。 除了经幡，沿途常有观景胜处：文成公主和亲的旧址，达坂山的草地雪山，高耸的雪峰，清凉的冰沟林海，油菜花初绽的门源……除此之外，永不重复的山川袒露着平缓的曲线，山峰顶的雪尖儿仿佛就在眼前，还有缀满草地的牲畜，弯弯曲曲的公路与流水，逆流的湟鱼，探头探脑的兔鼠，还有宝石尸体一般的、永远陈列着的青海湖。 匈奴人，西夏人，藏族人，回族人，汉族人。赶着牛羊的人，骑着马的人，驱着车的人，守着城的人，信教的人，征战的人，屠灵的人。 他们都说，祁连是天境。   # 西北，西北 再回西宁，这座城市由从前几日的错乱，回归熟悉，甚至亲切。那些裹着头巾卖酸奶的美丽少妇，那来来回回的莫家街与水井巷，那在舌尖嘟囔得飞快的方言，那实诚的物价，那繁忙但并不繁华的街市。早晚是清凉的空气，正午是强烈的太阳。路面常常在清洁和洒水，路边是种满各色花朵的围栏。 还是青海，不复印象中的贫穷与苍凉，甚至也并不浪漫。在西北的日子终于止于西宁。但我一直记得从前我如何地向往西宁，哪怕只是因为这个名字：西宁，西宁，在西北，安宁度日。 火车正往东开，驶离西北。一路上经过歌里的名字——兰州兰州，白银饭店，再往后是宁夏米粮川——每一站我都想下车，正如在青海，每条公路的路牌都有着致命的诱惑。每个地名都那么粗疏、俊朗，如一只骨节分明、手掌宽大的手。 我正在极不情愿地溜出它的掌心。多年前，我一再地尝试触碰的正是这只手。终于在今天，我也伸出手，想要将它挽留。
